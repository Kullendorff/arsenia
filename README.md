# Arsenia - Ars Magica Campaign Site

Character sheets and campaign journal for the Arsenia saga, an **Ars Magica** (5th edition) tabletop RPG campaign. Hosted via GitHub Pages.

## What's here

- Character sheet for Alain (stats, grimoire, background)
- Session journals and stories
- Campaign reference materials

## Tech

Static HTML/CSS with Python utility scripts for content management.
